A lightning-fast, fully-customisable bottom navigation bar for the mobile view of any WordPress site.

== Description ==
Streamline mobile UX with an attractive, sticky bottom menu. Seven free presets, WooCommerce cart count, live customiser, RTL, WPML, Instant-Articles and more.  
Upgrade to Pro for sub-menus, role rules, five extra styles, custom icon uploads, wishlist and premium support.

== Installation ==
1. Upload the `wp-mobile-nav-buttons` folder to `/wp-content/plugins/`.  
2. Activate through *Plugins → Installed Plugins*.  
3. Visit *Settings → Mobile Nav Buttons* to configure.
